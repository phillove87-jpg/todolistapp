## WF-01. 로그인 페이지 (`/login`)

### 화면 설명

- **목적:** 이메일·비밀번호로 사용자를 인증하고 JWT 토큰을 발급받아 서비스에 접근한다.
- **접근 조건:** 누구나 접근 가능. 이미 로그인된 상태에서 접근하면 `/`로 리다이렉트.
- **컴포넌트:** `LoginPage.jsx` → `LoginForm` 내부 폼
